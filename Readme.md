# Hands-on Perception

This repository contains the code for the **Hands-on Perception** course, featuring four comprehensive labs designed to build practical skills in perception systems.

## Course Overview

The course includes the following labs:

1. **Lab 1** - Introduction to perception fundamentals
2. **Lab 2** - Sensor data processing
3. **Lab 3** - Computer vision techniques
4. **Lab 4** - Advanced perception applications
